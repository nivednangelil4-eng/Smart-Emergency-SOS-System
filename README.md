# Project Title
Smart Emergency SOS System

# Overview of the Project
This project is a console-based Smart Emergency SOS System developed using Python. It simulates an emergency alert system where, upon user input, alerts are sent to predefined contacts.

The system also provides safety instructions to guide users during emergencies. While this version uses simulated outputs (printed messages), it demonstrates the core idea of how real SOS systems function.

# Features
* Send SOS alert to multiple contacts (simulated).
  
* Displays a mock location.

* Provides emergency safety tips.
  
* Menu-driven program (user-friendly).
 
* Input validation for incorrect choices.
 
* Continuous running system until user exits.

# Technologies/Tools Used
* Python – Main programming language.

* Built-in Functions – print(), input(), loops, and conditionals.
 
* Data Structures – List (cont) to store emergency contacts.

# Instructions for Testing
* Launch the Program
  
  * Run the Python file.
  * You will see the main menu.
    
* Test SOS Alert Option
  
  * Enter 1.
  * Check if:

    * Alert messages are printed for all contacts.
    * Location is displayed.
    * Safety tips appear.
      
* Test Safety Tips Option

  * Enter 2.
  * Verify all instructions are displayed correctly.
* Test Exit Option
  * Enter 3.
  * Program should terminate properly.

* Test Invalid Input

  * Enter wrong numbers (e.g., 5, -1).
  * Ensure error message appears.

* Test Loop Functionality

  * Enter 'y' to continue.
